# 简介
**概念：** 满足马尔可夫性质（无记忆性）的离散时间随机过程。
**核心：** **未来仅依赖当前状态，与历史状态无关**
# 定义与数学基础
## 核心定义：离散时间马尔可夫链
设随机过程 {$X_n​,n=0,1,2,…$}，状态空间 S 为**有限集或可列集**（记 $S={s_1​,s_2​,…,s_m​}$）。若对任意时刻 $n≥0$、任意状态 $i,j,i_0​,i_1​,…,i_{n−1}​∈S$，满足：

$$P(X_{n+1}​=j∣X_n​=i,X_{n−1}​=i_{n−1}​,…,X_0​=i_0​)=P(X_{n+1}​=j∣X_n​=i)$$
则称$X_n$为**离散时间马尔可夫链**。
- $S$：状态空间
	系统所有可能状态的集合
- $p_{ij}​$：一步转移概率
	定义：从状态 $i$ 经**一步**转移到状态 $j$ 的概率：
	$p_{ij}​=P(X_{n+1}​=j∣X_n​=i),∀i,j∈S$
- $P$：转移矩阵
	将 $p_ij​$ 排列为矩阵：
	$$P=\left\{
	\begin{align}
	p_{11}\ \ ​p_{12}​\ \ &​...\ \ ​p_{1m} ​​\\ 
	p_{21}\ \ ​p_{22}​\ \ ​&...\ \ ​p_{2m} ​​\\ 
	...\ \ ...​\ \ &​...\ \ ...\\
	p_{m1}\ \ ​p_{m2}​\ \ ​&...\ \ ​p_{mm} 
	\end{align}
	\right\}$$
	每行和为 1，称为**随机矩阵**。
- $π^{(0)}$：初始分布
	初始时刻（$n=0$）各状态的概率分布：
	$$π^{(0)}=(π_1^{(0)}​,π_2^{(0)},…,π_m^{(0)}​),∑_{i=1}^m​π_i^{(0)}​=1$$

其中 $π_i^{(0)}​=P(X_0​=i)$。
