# 随机过程
随机现象在某时刻$t$的取值是一个向量随机变量，用$X_t$表示，所有可能的状态组成状态集合$X$随机现象便是状态的变化过程。在某时刻t的状态$X_t$通常取决于t时刻之前的状态。
# 马尔可夫性
**定义**：给定系统当前状态，未来状态的概率分布与过去的历史无关。
设随机过程 $\{X_t, t \in T\}$ ，若对任意时刻$t_1 < t_2 < \cdots < t_n$均有
$$
P(X_{t+1}|X_{t_1},X_{t_2},\cdots,X_{t_n})=P(X_{t+1}|X_{t_n})
$$
# 马尔可夫过程
具有马尔可夫性的随机过程，我们通常使用元组$<\mathcal{X},\mathcal{P}>$，来描述一个马尔可夫过程
## 状态空间$\mathcal{S}$
系统所有可能状态的集合。
## 状态转移矩阵
假设一共有$n$个状态，此时$X=\{x_1,x_2,\cdots,x_n\}$，状态转移矩阵定义了所有状态$\mathcal{X}$对之间的转移概率，即
公式：
$$
\mathcal{P}=\left[
\begin{align}P(x_1|x_1)&\ \  ...\ \ P(x_n|x_1)\\
⋮\ \ \ \ \ \ &\ \ ⋱\ \  \ \ \ \ \ \ \ \ ⋮\\
P(x_1|x_n)&\ \  ...\ \ P(x_n|x_n)\end{align}
\right]
$$
- $P(x_j|x_i)$,一步转移概率:从状态$x_i$经**一步**转移到状态 $x_j$ 的概率
	从某个状态出发，到达其他状态的概率和必须为 1，即状态转移矩阵$\mathcal{P}$的每一行的和为 1。
**终止状态**（terminal state）
若一个状态没有概率传递给其他状态，它永远以概率 1 转移到自己，那我们称这个状态为**终止状态**（terminal state）。
**采样（sampling）**
给定一个马尔可夫过程，从某个状态出发，根据它的状态转移矩阵生成一个状态**序列（episode）**，这个步骤也被叫做**采样（sampling）**
# 马尔可夫奖励过程
在马尔可夫过程的基础上加入奖励函数$r$和折扣因子$\gamma$，就可以得到**马尔可夫奖励过程**。
马尔可夫奖励过程$<\mathcal{S},\mathcal{P},r,\gamma>$：
- $\mathcal{S}$是有限状态的集合。
- $\mathcal{P}$是状态转移矩阵。
- $r$是奖励函数，某个状态的奖励 指转移到该状态时可以获得奖励的期望。
- $\gamma$是折扣因子（discount factor），的取值范围为。引入折扣因子的理由为远期利益具有一定不确定性，有时我们更希望能够尽快获得一些奖励，所以我们需要对远期利益打一些折扣。接近1的更关注长期的累计奖励，接近 0 的更考虑短期奖励。
## 回报
在一个马尔可夫奖励过程中，从第$t$时刻状态$S_t$开始，直到终止状态时，所有奖励的衰减之和称为**回报**$G_t$（Return），公式如下：
$$
G_t= R_t+\gamma R_{t+1}+\gamma ^2R_{t+2}+...=\Sigma_{k=0}^\infty \gamma ^kR_{t+k}
$$
- $R_t$：在$t$时刻获得的奖励。
## 价值函数
在马尔可夫奖励过程中，一个状态的期望回报（即从这个状态出发的未来累积奖励的期望），被称为这个状态的**价值**（value）
