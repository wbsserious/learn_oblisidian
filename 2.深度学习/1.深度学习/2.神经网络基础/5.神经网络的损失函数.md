# 二分类问题
Binary Cross Entropy（BCE）——二进制交叉熵
公式：
$$
loss=-y\log{\hat{y}}-(1-y)\log{(1-\hat{y})}
$$
- 配合：**Sigmoid** 输出（0~1）
- 用途：二分类、检测、分割二分类
# 多分类问题
Cross Entropy Loss（交叉熵）——交叉熵损失
公式：
$$
CE =-\sum_i{\hat{y}_i}log(softMax(f(x_i)))
$$
$f(x_i)$代表的是预测之后的分数
- 配合：**Softmax** 输出
- 用途：图像分类、文本分类、语音分类

# 回归任务
## L1 Loss（MAE，平均绝对误差）
**使用的比较少**
公式
$$
L1=\frac{1}{N}\sum|y-\hat{y}|
$$
![[Pasted image 20260208174401.png]]
- 优点：对**异常点不敏感**，梯度稳定
- 缺点：0 点不可导，收敛后期抖动
## L2 Loss（MSE，均方误差）【最常用】
nn.MSE loss
$$
MSE=L1=\frac{1}{N}\sum(y-\hat{y})^2
$$
![[Pasted image 20260208174556.png]]
- 优点：**处处可导、优化稳定、收敛快**
- 缺点：**对异常点非常敏感**

- 绝对值小 → 用 L2（平滑）

# 总结
- **回归**：默认 **MSE**，有异常点 → **Smooth L1/Huber**
- **二分类**：**BCEWithLogitsLoss**
- **多分类**：**CrossEntropyLoss**
- **大模型 NLP**：**CrossEntropy + 可选 LabelSmoothing**
- **样本不平衡**：**Focal Loss / Dice Loss**
- **自监督 / 对比学习**：**InfoNCE**
- **分割**：**Dice + CE 混合**
- **GAN/VAE**：对抗损失 + 重建损失 + KL
