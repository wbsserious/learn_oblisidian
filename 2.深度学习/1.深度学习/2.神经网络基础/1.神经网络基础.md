# 神经网络层级机构
**层级结构**核心是：**分层堆叠神经元，层间全连接、层内无连接**。
从输入到输出逐层传递、提取特征。

# 最基础神经网络结构——三层感知机（标准前馈网络）
1. **输入层（Input Layer）**
- 最第一层，负责接收原始数据
- 节点数 = 特征维度（比如图片像素、表格列数）
- 只做**传递**，不做计算、不激活。
2. **隐藏层（Hidden Layer）**
- 中间的层，看不见输出，负责**特征提取 / 学习**
- 可以有 **1 层～N 层**（层数多了就是 “深度神经网络”）
- 每层内部：多个神经元，做加权求和 + 激活函数
- 层与层之间：**全连接（Dense/FC）**，每个神经元连下一层所有神经元
- **内部状态值**:神经元接收到的输入（上一层的输出）的加权求和结果
- **激活值**:激活函数对内部状态值进行非线性变换后得到的结果
3. **输出层（Out Layer）**
- 最后一层，输出预测结果
- 节点数 = 任务需要（任务数目，回归一个值）
- 携带对应的激活函数。 
**神经网络参数数量：=隐藏层每层神经元*（权重+偏执）+输出层神经元*（权重+偏执）**
### ***注意***
- **隐藏层激活函数**：负责**引入非线性**，让网络能学复杂规律（核心是 “拟合能力”）
- **输出层激活函数**：负责**把输出映射到任务需要的范围 / 概率**（核心是 “任务匹配”）
# 常见神经网络的层级结构
1. CNN（卷积神经网络）
	输入层 → 卷积层 Conv → 池化层 Pool → 卷积层 → 池化 → 全连接层 → 输出层
2. RNN/LSTM/GRU（看序列、文本、时间序列）
	输入层 → 循环层（带记忆）→ 全连接 → 输出层
3. Transformer（大模型、GPT）
	嵌入层 → 多头注意力层 → 前馈层 → 归一化 / 残差 → 堆叠 N 次 → 输出层
