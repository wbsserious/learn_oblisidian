# 神经网络常用优化器（损失函数优化器）分类与详解
优化器的核心作用是**根据损失函数的梯度更新模型参数**，最小化损失。按**梯度阶数 / 优化思想**分为**一阶优化器、二阶优化器、自适应学习率优化器、新型启发式优化器**四大类，覆盖深度学习全场景。
## 一、基础一阶优化器（无自适应 / 动量）
仅依赖**一阶梯度**，无历史梯度累积，结构最简单。
1. **SGD（随机梯度下降）**
    - 原理：单个 / 小批量样本梯度直接更新参数，`θ = θ - η·∇L(θ)`
    - 特点：实现简单、泛化性好；梯度震荡大、收敛慢，病态曲面难收敛
    - 适用：CV 分类 / 检测等大部分场景，搭配动量使用
2. **SGD+Momentum（动量 SGD）**
    - 原理：引入**动量项**累积历史梯度，平滑梯度震荡，加速收敛
    - 特点：冲出局部最优、平坦极小值，解决 SGD 抖动问题
    - 适用：主流 CV 任务，深度学习最经典组合
## 二、带自适应学习率的一阶优化器（主流首选）
自动为不同参数分配学习率，解决手动调参难题，工业界最常用。
1. **AdaGrad**
    - 原理：累积历史梯度平方做分母，**稀疏特征梯度学习率自动衰减**
    - 缺点：梯度持续累积，后期学习率过小导致训练停滞
    - 适用：NLP 稀疏特征、推荐系统
2. **RMSProp**
    - 原理：梯度平方**指数滑动平均**，解决 AdaGrad 累积衰减问题
    - 特点：稳定适配非凸优化，是 Adam 的前身
    - 适用：RNN/LSTM 时序任务
3. **Adadelta**
    - 原理：进一步移除学习率超参，用参数更新量的滑动平均归一化
    - 特点：无需手动调学习率，稳定性强
4. **Adam（自适应矩估计）**
    - 原理：融合**Momentum (一阶矩)** + **RMSProp (二阶矩)**
    - 特点：收敛极快、鲁棒性强，默认超参适配多数场景
    - 缺陷：可能收敛到局部最优，泛化性略逊 SGD
    - 适用：快速迭代实验、NLP/CV/ 多模态通用场景
5. **AdamW**
    - 原理：Adam 的**权重衰减修正**，分离权重衰减与 L2 正则
    - 特点：解决 Adam 权重衰减失效问题，泛化性大幅提升
    - 适用：Transformer、大语言模型、CV 预训练
6. **AdamP**
    - 原理：针对 Adam 的权重归一化优化，改善小模型 / 卷积网络性能
7. **NAdam**
    - 原理：Adam+Nesterov 动量，提前感知梯度趋势，收敛更快
## 三、二阶优化器（拟牛顿 / 海森近似）
利用**二阶曲率信息**，收敛速度远快一阶，但计算 / 显存成本高。
1. **L-BFGS**
    - 拟牛顿法主流实现，不存储完整海森矩阵，仅保留历史梯度向量
    - 适用：小参数量网络、全批次训练、模型精调收尾
2. **BFGS/SR1**
    - 经典拟牛顿算法，高维参数下不可用
3. **K-FAC/Shampoo**
    - 海森矩阵低秩分解 / 分块近似，适配中等规模模型
    - 适用：大模型二阶优化研究、高精度收敛场景
## 四、新型启发式 / 大模型优化器
针对大模型、Transformer 架构设计，训练效率 / 稳定性更优。
1. **Lion**
    - 原理：基于**sign (梯度)** 更新，移除二阶矩计算
    - 特点：显存占用低、训练速度快，大模型主流新秀
    - 适用：LLM、AIGC、大规模预训练
2. **Sophia**
    - 原理：结合海森对角近似的自适应优化器，大模型收敛更快
3. **AdaHessian**
    - 自适应二阶优化，用随机梯度近似海森，降低计算开销
## 五、优化器核心对比表

|优化器|核心思想|收敛速度|显存占用|泛化性|主流场景|
|---|---|---|---|---|---|
|SGD|纯梯度下降|慢|低|优|最终精调、CV|
|SGD+Momentum|动量 + 梯度|中|低|优|CV 经典训练|
|Adam|一阶 + 二阶矩|快|中|良|通用快速实验|
|AdamW|Adam + 权重衰减修正|快|中|良|Transformer / 大模型|
|Lion|梯度符号更新|极快|低|良|LLM/AIGC 预训练|
|L-BFGS|拟牛顿二阶|极快|中高|优|小模型全批次精调|

## 六、选型建议
1. **快速实验 / 新手**：直接用 **AdamW**，默认超参基本够用
2. **CV 卷积网络**：**SGD+Momentum** 泛化性最优，最终精调必选
3. **大模型 / Transformer/LLM**：**Lion/AdamW**，兼顾速度与显存
4. **小模型 / 全批次拟合**：**L-BFGS** 快速收敛到最优解
5. **稀疏特征 / 推荐系统**：**AdaGrad/RMSProp**
