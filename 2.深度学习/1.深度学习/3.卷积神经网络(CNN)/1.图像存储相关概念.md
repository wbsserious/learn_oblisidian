图像由多个灰度图叠加形成的
pytorch 支持的图片格式
|C|H|W|
|----|----|----|
|通道数|高度|宽度|
___
卷积、归一化、预处理，**都要先搞清楚通道顺序和数量**。
使用matplotlib导入图像：
```
import matplotlib.pyplot as plt
plt.iread("图片路径名")
```
**在pytorch中图片的格式需要调整为：CHW**
**通过matplotlib默认导入的图片格式为：HWC**
matplotlib展示数据的时候，对图片格式的要求为：HWC
在读取，处理，展示数据之间需要对应相应的数据格式转换。
```
1. matplotlib先读取文件
import matplotlib.pyplot as plt
img=plt.iread("图片路径名")
2. 图片存储格式转变为np.array格式
import numpy as np
img_array = np.array(img)
3. 转换为torch的格式
import torch
img_torch = torch.from_numpy(img_array)
4. 查看图片格式，将图片通道数转换为对应格式
print(img_torch.shape)
img_torch.permute(2,0,1)
print(img_torch.shape)
HWC转换为CHW
```
图片存储的方式
```
plt.imsave(img)
```
