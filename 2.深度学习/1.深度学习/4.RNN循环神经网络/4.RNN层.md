## 整个RNN网络的计算过程
![[Pasted image 20260226003041.png]]
词嵌入层:
	把token 转为 词向量
RNN层：
	循环计算隐藏状态h
全连接层:
	输出当前时间步的预测结果y
隐藏状态作用：
1.记忆功能; 2.上下文理解;3.连接不同时间步
## RNN层/RNN单元的计算过程
循环计算隐藏状态h
公式：
$$
h_t=tanh(W*h_{t-1}+V*{x_t}+b_t)
$$
输入(h_{t-1})：上个时间步的隐藏状态+当前时间步的词向量
输出(h_t)：当前时间步的隐藏状态
隐藏状态作用：
1.记忆功能; 2.上下文理解;3.连接不同时间步

| 符号     | 核心含义                            | 通俗解释（结合例子）                       |
| ------ | ------------------------------- | -------------------------------- |
| ht​    | 第 t 个时间步的**隐藏状态**（Hidden State） | t=2 时，h2​就是输入 “爱” 后，RNN 的 “当前记忆” |
| tanh() | 双曲正切激活函数（值域 [-1,1]）             | 把计算结果 “压缩” 到 [-1,1]，避免数值爆炸       |
| W      | 隐藏层循环权重矩阵（Weight for Hidden）    | 控制 “上一时刻记忆” 对当前的影响（所有时间步共享）      |
| h_t−1​ | 第 t-1 个时间步的隐藏状态（上一时刻的记忆）        | t=2 时，h1​是输入 “我” 后的 RNN 记忆       |
| V      | 输入层到隐藏层的权重矩阵（Weight for Input）  | 控制 “当前输入” 对记忆的影响（所有时间步共享）        |
| x_t​   | 第 t 个时间步的输入（通常是词嵌入后的向量）         | t=2 时，x2​是 “爱” 对应的词向量            |
| b      | 偏置项（Bias）                       | 给计算结果加一个 “偏移量”，让模型更灵活（所有时间步共享）   |
|        |                                 |                                  |

**h_t**被传递到两个地方，**全连接层**和**下一个时间步的RNN单元**
