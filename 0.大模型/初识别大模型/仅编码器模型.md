#  BERT
**双向编码器表示模型**
Bidirectional Encoder Representations from Transformers
BERT is an encoder-only architecture that focuses on representing language
BERT 是一个只有编码的模型。它值用来解释语言。
This means that it only uses the encoder and removes the decoder entirely.
这意味着它完全只是用编码器，完全移除了解码器部分。
These encoder blocks are the same as we saw before: self-attention followed by feedforward neural networks. 
The input contains an additional token, the [CLS] or classification token, which is used as the representation for the entire input.
输入中额外加入了一个特殊 token——**[CLS]（分类 token）**，它被视作**整个输入的表示**。  
Often, we use this [CLS] token as the input embedding for fine-tuning the model on specific tasks, like classification.
通常，我们把该 [CLS] token 的嵌入向量拿来微调模型，以完成分类等下游任务。
## masked language modeling
**掩码语言建模**（简称 MLM）
Training these encoder stacks can be a difficult task that BERT approaches by adopting a technique called masked language modeling.
训练这些编码器堆栈是一项艰巨任务，BERT 通过采用一种称为**掩码语言建模**（masked language modeling）的技术来应对。
this method masks a part of the input for the model to predict. This prediction task is difficult but allows BERT to create more accurate (intermediate) representations of the input.
