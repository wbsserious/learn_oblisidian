# MSE、MAE、RMSE 的定义及区别是什么？#
**MSE**：均方误差
**MAE**：平均绝对误差
**RMSE**： 均方根误差
区别：MAE 不能体现出误差大的数据点，RMSE放大大误差的数据点对指标的影响， 但是对异常数据比较敏感

# 梯度下降的常见类型有哪些？各自的特点是什么？#
批量梯度下降算法 BGD
使用全量数据进行计算，特点为训练速度较慢。
随机梯度下降算法 SGD
简单，高效，不稳定。SG每次只使用一个样本迭代，若遇上噪声则容易陷入局部最优解。
mini——批量梯度下降算法 MBGD
结合了 BGD 的稳定性和 SGD 的速度，**是目前深度学习中最常用的方法**
随机平均梯度下降算法 SAG 
核心思路是通过缓存每个样本的最新梯度、迭代时仅更新单样本梯度并取全缓存平均，实现方差缩减。
初期训练速度较差。

# 梯度下降中，学习率（步长）的作用是什么？设置不当会有什么问题？#
**学习率（Learning Rate）**，决定了每次迭代步长的大小
- **学习率太小**：收敛速度极慢，需要迭代很多次才能到达最低点，训练时间长。
- **学习率太大**：可能会 “迈过” 最低点，导致在最低点附近震荡，甚至发散（无法收敛）。

# L1 正则化（Lasso 回归）和 L2 正则化（Ridge 回归）的区别是什么
L1正则化：惩罚项为所有权重的绝对值之和。
L2正则化：惩罚项为为所有权重的平方和。
区别：L1正则化可以让权重变为零，L2正则化会让所有权重都变得**很小**，但不会让它们变为 0
