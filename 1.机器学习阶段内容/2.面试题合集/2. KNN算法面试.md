#  请简述 KNN 算法的核心思想
KNN算法为一种惰性学习的非参数的非监督学习算法。它的核心思维是，相似样本在特征空间中彼此靠近。

#  KNN 算法处理分类问题和回归问题的流程分别是什么？
分类问题：
1.计算样本中所有样本距离位置样本的距离。
2.将训练样本根据距离进行排序。
3.取出距离最近的前K个值的数据。
4.找出K个值中出现频率最高的类别。
5.该类别就是未知样本的预测值。
回归问题：
1.计算样本中所有样本距离未知样本的距离。
2.将训练样本根据距离大小进行排序。
3.取出距离最近的前K个值的数据。
4.根据这K个样本的标签值取平均值。
5.未知样本的标签就是该平均值。

#  KNN 算法中，K 值的选择对模型有什么影响？如何选择合适的 K 值？
K值决定了邻居数量，K值过大容易出现欠拟合，K值过小容易出现过拟合。

#  为什么需要特征预处理？归一化和标准化的区别是什么？分别适用于什么场景？
- 消除量纲（单位）影响
- 提升模型收敛速度
- 改善模型精度与稳定性
- 满足特定算法的输入要求
归一化：通过对原始数据进行变换把数据映射到指定区间之间。
标准化：通过对原始数据进行标准化，转换为均值为0标准差为1的标准正态分布的数据

#  交叉验证和网格搜索的作用是什么？两者结合的优势是什么？
**交叉验证法**是划分数据集的一种方法，目的就是为了得到更加准确可信的模型评分。
**网格搜索**对指定的超参数组合（笛卡尔积）逐一测试，选最优
## 两者结合的优势##
•交叉验证解决模型的数据输入问题(数据集划分)得到更可靠的模型
•网格搜索解决超参数的组合
•两个组合再一起形成一个模型参数调优的解决方案
