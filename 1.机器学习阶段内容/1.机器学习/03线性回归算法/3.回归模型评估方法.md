# 平均绝对误差 Mean Absolute Error MAE #
![[Pasted image 20260120215411.png]]
n 为样本数量, y 为实际值, y ̂为预测值
MAE 越小模型预测约准确
MAE 反应的是“真实”的平均误差
**特点**：
- **鲁棒性（Robustness）**：对异常值（Outliers）不敏感。因为它取绝对值，单个极大的误差不会像平方那样被放大。
- **直观性**：单位与原始数据一致（例如预测房价，MAE=5 万，意味着平均差 5 万），容易向业务方解释。
- **数学缺点**：在零点不可导（绝对值函数的尖角），在使用梯度下降优化时不如 MSE 方便（虽然现在有次梯度方法解决）。
# 均方误差 Mean Squared Error MSE #

![[Pasted image 20260120220553.png]]
n 为样本数量, y 为实际值, y ̂为预测值
MSE 越小模型预测约准确
**特点**：
- **最常用**：是线性回归（Linear Regression）默认的损失函数。
- **敏感性**：对异常值**非常敏感**。因为误差被平方了，一个很大的误差会主导整个损失函数。
- **数学优点**：函数处处可导且光滑，非常适合使用梯度下降算法进行优化。
- **缺点**：单位是原始数据的平方（例如房价差的平方），缺乏直观的物理意义。
# 均方根误差 Root Mean Squared Error (RMSE)#
![[Pasted image 20260120220635.png]]
n 为样本数量, y 为实际值, y ̂为预测值
RMSE 越小模型预测约准确
**特点**：
- **还原单位**：它保留了 MSE “惩罚大误差” 的特性，但把单位还原回了原始数据（例如房价差），兼具了数学特性和业务解释性。
- **衡量准确度**：通常是评估回归模型**准确度**的首选指标。

**RMSE > MAE都能反应真实误差，但是RMSE会对异常点更加敏感**
