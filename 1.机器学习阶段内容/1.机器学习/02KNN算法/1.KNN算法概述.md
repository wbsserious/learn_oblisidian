K-近邻算法（K Nearest Neighbor，简称KNN）
**定义**：KNN 是一种**惰性学习（Lazy Learning）** 和**非参数（Non-parametric）** 的**监督学习**算法。
**核心特点** ：
- 非参数：不对数据分布做假设，适配非线性数据。
- 惰性学习：训练无显式模型拟合，仅存储样本；预测时计算距离，适合数据频繁更新场景。
- 基于实例：直接用训练样本推理，无需抽象模型。
**算法思想**：如果一个样本在特征空间中的 k 个最相似的样本中的大多数属于某一个类别，则该样本也属于这个类别。
相似样本在特征空间中彼此靠近。
### 三大关键要素

| 要素   | 说明                             | 常见选择                                                                   |
| ---- | ------------------------------ | ---------------------------------------------------------------------- |
| K 值  | 决定邻居数量，影响偏差与方差。K 过小易过拟合，过大易欠拟合 | 常用 1、3、5、7 等奇数；通过交叉验证（如 5 折 / 10 折）选最优值                                |
| 距离度量 | 衡量样本相似度，决定 “近邻” 标准             | 欧氏距离（连续特征）、曼哈顿距离（高维 / 稀疏）、闵可夫斯基距离（通用）、余弦相似度（文本 / 向量）参考[[2.距离度量]] |
| 决策规则 | 分类用多数表决，回归用均值 / 加权均值           | 分类：多数票胜出；回归：简单平均或按距离加权（近邻权重更高）                                         |
|      |                                |                                                                        |
### 分类问题的解决流程###
1.计算样本到每一个训练样本的距离。
2.将训练样本根据距离大小升序排列。
3.取出距离最近的K个训练样本。
4.进行多数表决，统计K个样本中那个类别的样本个数最多。
5.将未知的样本归属到出现次数最多的类别
### 回归问题的解决流程###
1.计算未知样本到每一个样本的距离。
2.将训练样本根据距离大小进行排序。
3.取出距离最近的K个训练样本。
4.计算这K个样本的平均值。
5.将平均值当成未知样本的
