**算法概念**：聚类（Clustering）是**无监督机器学习**的一类核心算法，它在**没有预先标注的类别标签、没有人工指定的分类规则**的前提下，仅依据数据样本自身的特征分布与相似性度量，按照**簇内相似度最大化、簇间相似度最小化**的原则，将数据集自动划分为若干互不相交的子集，每个子集称为一个**簇（Cluster）**。
无监督学习算法，主要用于将相似的样本自动归到一个类别中，计算样本和样本之间的相似性，一般使用欧式距离。
## **聚类与分类的区别**

| 对比维度 | 聚类                      | 分类                   |
| :--- | :---------------------- | :------------------- |
| 学习类型 | **无监督学习**               | **有监督学习**            |
| 数据要求 | 无标签数据，仅用特征              | 带有真实标签的标注数据          |
| 目标   | 发现数据的潜在结构、自动分组          | 学习从特征到标签的映射，预测未知样本类别 |
| 输出结果 | 簇（无预定义含义，需要人工解释）        | 预定义的类别（有明确语义）        |
| 典型算法 | K-Means、DBSCAN、层次聚类、GMM | 决策树、SVM、逻辑回归、集成学习    |
