**核心思想**：基学习器**顺序生成**，**后一个学习器依赖于前一个学习器的结果**；每个新的学习器都会针对性地修正前序学习器的错误，从而降低集成模型的**偏差**，提升模型的拟合能力。
每一轮都针对上一轮没做好的地方重点优化，层层叠加，最终得到一个强预测模型
**构建思路**：
- **初始状态**
	一开始给所有训练样本分配相同的权重，训练第一个基础模型。
- **迭代优化（串行核心）**
	每训练完一个基学习器，就计算它的预测误差：
	- 对**分类任务**：记录分错的样本，**提高这些样本的权重**，让下一个模型更关注它们；
	- 对**回归任务**：计算当前模型的**预测残差（真实值 - 预测值）**，让下一个模型不去拟合原始标签，而是直接拟合上一轮的残差。
- **顺序依赖，无法完全并行**
	第 k 个模型必须等第 k−1 个训练完成、误差 / 权重计算完毕才能开始，整个流程是链式依赖，天然不支持并行加速。
- **最终融合**
	一般是对所有基学习器的结果做**加权求和**，权重由每个基模型的预测精度决定：效果越好的模型，权重越高。
# 实现方式：AdaBoost 与梯度提升类
# AdaBoost（自适应提升）
**核心思想**：
**自适应调整样本权重**，通过不断提高错分样本的权重，让后续弱分类器重点学习难分样本。
AdaBoost 是**最经典的串行集成算法**，主打样本权重调整，多用于分类任务。
**构建过程**
- 初始化所有样本权重一致。
- 训练一个弱学习器。
- 计算该学习器的误差率，误差率越低模型权重越高。
- 更新模型权重：分错的样本权重提升，构建正确的样本权重降低。
- 重复训练下一个学习器，直到到达预定的轮数。
- 最终按照各弱分类器的权重加权投票获得最终结果。
特点：
- 对异常值和噪声**非常敏感**，因为错分样本会被不断放大权重；
- 只能处理分类，原生不支持回归；
- 工程上逐渐被梯度提升树替代。
# 梯度提升类算法
**核心思路**：
**把集成训练转化为梯度下降优化**，每一棵新树不再直接拟合标签，而是拟合**损失函数的负梯度（残差的推广形式）**
**拟合残差 + 梯度下降**
# GBDT（梯度提升决策树）
**核心思想**：
**多棵树依次训练，每一棵新树都不去拟合原始标签，而是去拟合前面所有树的 “预测误差（残差 / 负梯度）”，最终把所有树的结果加起来得到最终预测**。
梯度提升类算法的**基础原型**。
构建思路：
1. 初始化弱学习器。
	- 在以平方损失（MSE）为损失函数的情况下，最优常数为样本标签的均值。
2. 构建基模型学习器，每一个基模型学习器拟合上一个学习器的负梯度。
	1. 对每个样本计算当前模型预测的负梯度。**平方损失时（MSE）负梯度等价于残差**

	2. 用 CART 回归树拟合残差残差。
	3. 确定每片叶子的输出值。
	4. 更新模型预测结果。
	 ![[Pasted image 20260129203026.png]]
3. 直到达到指定的学习器个数
4. 获取最终模型。
	- 当输入未知样本时，将所有弱学习器的输出结果组合起来作为强学习器的输出

### 分类任务的差异
分类任务只是**损失函数和最后映射方式不同**，训练树的过程仍然是拟合负梯度：
- 二分类：使用逻辑损失（交叉熵），最终通过 Sigmoid 映射为类别概率；
- 多分类：训练多组并行的 GBDT，通过 Softmax 输出各类概率。
    本质上，树的训练依然是**回归任务**。
## GBDT 的核心特点
### 1. 严格串行，无法并行训练
- 每一棵树必须等上一轮的残差 / 梯度计算完成才能训练；
- 只能在单棵树内部做有限并行，整体训练效率远低于随机森林。
### 2. 拟合能力极强，偏差极低
- 通过迭代修正误差，能捕捉高度复杂的非线性关系；
- 在结构化表格数据上，精度普遍优于传统模型与单一决策树。
### 3. 天然处理特征交互与非线性
- 决策树本身支持特征组合、分段非线性，GBDT 无需手动做特征交叉、归一化。
### 4. 可输出特征重要性
- 基于特征在分裂时带来的损失下降量，统计全局重要程度，用于特征筛选。
### 5. 更容易过拟合
- 持续拟合误差会把噪声也学进去，需要严格正则化控制。
## GBDT 的优点
1. 预测精度高。
2. 适用任务广泛。
3. 数据预处理成本低。
4. 可解释性较强。
5. 理论完备、易于扩展。
## GBDT 的缺点
1. 训练速度慢
2. 对异常值敏感
3. 调参成本高
4. 不适合高维稀疏数据
	- 对文本、One‑Hot 类稀疏特征表现不佳，远不如线性模型与深度学习。
5. 原生不支持类别特征
# XGboost（极端梯度提升）
**核心思想**：在 GBDT 只使用**一阶导数（梯度）** 的基础上，增加**二阶导数（海森矩阵）**，让损失函数逼近更精准；同时加入正则项、缺失值自动处理、并行优化、缓存感知等工程设计。

## 构建思路
- **初始化**
    用常数（通常是标签均值 / 对数几率）作为初始预测 $y^​i(0)$​。
- **迭代构建每一棵树**
    - 对当前样本，计算**一阶导数 $g_i$​ 和二阶导数 $h_i​$**。
    - 遍历特征，寻找最优分裂点，计算分裂增益，增益必须大于 γ 才允许分裂。
    - 缺失特征会自动被分到左子树或右子树，选择增益更大的方向。
    - 生成树结构后，计算每个叶子的**最优权重 wj​**。
    - 乘以学习率 η，更新模型预测 $y^​{i(t)​}=y^​i(t−1)​+ηft​(xi​)$。
- **终止条件**
    达到设定的树数量 `n_estimators`，或分裂增益为负、损失不再下降。
- **预测**
    所有树的输出加权累加，分类任务经过 Sigmoid / Softmax 映射为概率。
