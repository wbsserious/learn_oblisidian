# THULAC
- **定位**：清华大学出品的学术级中文分词 / 词性标注工具，主打**高精度**。
- **核心优势**：
    - 分词 + 词性标注精度极高，尤其适合书面语、学术文本；
    - 模型基于结构化感知机，对未登录词（新词）识别效果好；
    - 支持繁体中文分词；
    - 提供 Python/Java/C++ 多语言版本。
- **缺点**：
    - 速度较慢，比 Jieba 慢一个量级，不适合大规模文本处理；
    - 安装略繁琐（需下载预训练模型）；
    - 功能单一，仅支持分词 + 词性标注，无其他 NLP 能力。
- **适用场景**：学术研究、对精度要求高的小批量文本分析、专业领域文本处理。
# HanLP
- **定位**：全功能 NLP 工具包，分词只是其众多功能之一，覆盖从分词到句法分析、情感分析的全流程。
- **核心优势**：
    - 功能全面：支持分词、词性标注、命名实体识别（NER）、句法分析、文本分类、关键词提取等；
    - 精度高，且支持多语言（中 / 英 / 日）、多领域（通用 / 金融 / 法律）；
    - 提供 Python/Java 版本，Java 版本功能更完整；
    - 支持自定义模型、增量训练，适合企业级应用。
- **缺点**：
    - 体积大，依赖多，入门门槛比 Jieba 高；
    - 轻量场景下使用略显 “重”。
    
- **适用场景**：复杂 NLP 项目（如智能问答、文本挖掘）、企业级应用、需要多维度文本分析的场景。
