# RoPE（旋转位置编码，Rotary Positional Embedding）
**核心原理：**
对 Q/K 向量按位置做**旋转**，让注意力分数只由**相对距离**决定，不依赖绝对位置，且**无需学习参数**。
## 主流模型采用情况

| 模型系列                      | 位置编码     | 备注           |
| ------------------------- | -------- | ------------ |
| LLaMA 1/2/3               | **RoPE** | 开源标杆，全面采用    |
| GPT-4 / GPT-4o            | **RoPE** | 结合外推（如 YARN） |
| PaLM / Gemini             | **RoPE** | Google 主流方案  |
| ChatGLM / Qwen / InternLM | **RoPE** | 国产主流         |
| Claude 3                  | **RoPE** | Anthropic 方案 |
| T5/Flan-T5                | 可学习相对偏置  | 仅 T5 家族沿用旧方案 |
| BERT/GPT-2                | 可学习绝对嵌入  | 早期方案，已非主流    |

# ALiBi（Attention with Linear Biases）
- 特点：**无参数、极简、外推极强**，直接给注意力加距离惩罚。
- 代表：BLOOM、部分 LLaMA 变体、小模型 / 移动端场景。
- 地位：**次主流**，适合资源受限场景，但精度略逊 RoPE。

## 1. 真正的 **可学习位置编码**
- **代表**：BERT、GPT-2
- **形式**：`nn.Embedding(max_len, d_model)`
- **是否可学习**：**是，有参数，会被梯度更新**
- **加在哪里**：直接和词嵌入 **相加**
## 2. **有参数，但不叫 “可学习位置编码”**

- **代表**：T5 / Flan-T5 的 **相对位置偏置（Relative Position Bias）**
- **是否可学习**：**是，有参数，会更新**
- **但不是**传统意义上的位置编码
- **加在哪里**：加在 **注意力分数** 上，不是加在 embedding 上

## 3. **完全无参数、不可学习**（现在大模型主流）

- **代表**：**RoPE（LLaMA、GPT-4、Qwen、ChatGLM）**
- **代表**：**ALiBi（BLOOM）**
- **是否可学习**：**完全不可学习，无参数**
- **原理**：纯数学变换 / 纯规则
