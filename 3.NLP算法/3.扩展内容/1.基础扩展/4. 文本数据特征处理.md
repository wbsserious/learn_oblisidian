# 阶段 1：文本预处理（特征处理的前置步骤）
先完成文本清洗和标准化，为后续特征提取打基础：
1. **基础清洗**：去除乱码、特殊符号、HTML 标签、无关链接、重复文本；统一全角 / 半角、大小写；
2. **分词 / 切词**：
    - 中文：jieba、THULAC、pkuseg（分词 + 词性标注）；
    - 英文：nltk.word_tokenize、spaCy（分词 + 词形还原）；
    - 深度学习专用：BPE、WordPiece、Unigram（Subword 切分，适配 BERT 类模型）；
3. **噪声过滤**：去除停用词（如 “的 / 了 / 是”）、低频词（如出现次数 < 5 的词）、无意义字符；
4. **语言规范化**：
    - 英文：词形还原（Lemmatization）、词干提取（Stemming）；
    - 中文：繁简转换、错别字修正、同义词替换；
5. **句子 / 段落拆分**：按标点 / 语义拆分句子，适配序列模型输入。
# 阶段 2：传统机器学习（非深度）文本特征提取
适用于逻辑回归、SVM、随机森林等模型，核心是构建稀疏 / 稠密特征矩阵：
1. **基于词频的统计特征**
    - 词袋模型（BoW）：统计每个词在文本中是否出现（0-1）或出现次数（计数）；
    - TF-IDF：加权词频（TF：词在文本中频率，IDF：词在语料中稀有度） $TF-IDF = TF \times log(\frac{总文本数}{包含该词的文本数+1})$ 总文本数包含该词的文本数；
    - N-gram 特征：unigram（单字 / 单词）、bigram（双词）、trigram（三词），捕捉局部语序信息；
2. **基于词汇的语义特征**
    - 词嵌入平均（Word Embedding Average）：预训练词向量（Word2Vec/GloVe）取文本中所有词向量的均值 / 加权均值；
    - 主题特征：LDA/LSI/NMF 提取主题分布（如 10 个主题维度的概率分布）；
3. **结构化特征**
    - 长度特征：文本字符数 / 词数、句子数、平均句长；
    - 词性特征：名词 / 动词 / 形容词占比、特定词性词数；
    - 统计特征：标点符号数、情感词数、关键词数。
# 阶段 3：深度学习文本特征处理（核心）
适用于 CNN/RNN/Transformer 类模型，核心是构建序列 / 矩阵型数值特征
1. **文本向量化（Tokenization + 编码）**
    - 词典映射：构建词汇表（Vocab），将每个词 / Subword 映射为唯一整数 ID（如 “苹果”→1001）；
    - 序列标准化：
        - 截断（Truncation）：超过 max_seq_len 的文本截断（如 BERT 默认 max_seq_len=512）；
        - 填充（Padding）：不足 max_seq_len 的文本补 0（PAD）；
    - 特殊标记添加：CLS（分类任务）、SEP（句子分隔）、MASK（掩码）    
2. **词嵌入（Word Embedding）**
    - 静态嵌入：预训练 Word2Vec/GloVe/FastText，固定向量不参与训练；
    - 动态嵌入：模型内置嵌入层（随训练更新），或加载预训练嵌入后微调；
3. **上下文感知嵌入（语义特征核心）**
    - 预训练语言模型（PLM）：BERT/RoBERTa/ALBERT 等，输入 token ID 后，输出包含上下文信息的动态语义向量（如 CLS 位向量作为文本分类特征）；
    - 句子嵌入（Sentence Embedding）：Sentence-BERT、SimCSE，直接输出整句的稠密语义向量（适配分类 / 相似度任务）；
4. **序列特征增强**
    - 位置编码（Positional Encoding）：Transformer 模型专用，添加词的位置信息（正弦余弦编码 / 可学习位置嵌入）；
    - 注意力掩码（Attention Mask）：标记 PAD 部分，避免模型关注填充内容；
    - 特征融合：拼接文本长度、词性占比等统计特征到语义向量中，提升分类效果。
# 阶段 4：特征优化（通用环节）
对提取后的特征做进一步处理，提升模型效果：
1. **特征筛选**：去除低方差特征（如所有文本中都出现的词）、高相关特征；
2. **特征归一化 / 标准化**：
    - 传统特征：TF-IDF 特征做 L2 归一化（避免长度影响）；
    - 深度特征：语义向量做均值归一化；
3. **特征降维**：
    - 传统特征：PCA、LDA（降低 TF-IDF 稀疏矩阵维度）；
    - 深度特征：t-SNE/UMAP（可视化用）、知识蒸馏（压缩预训练模型特征）；
4. **特征增强**：   
    - 文本层面：同义词替换、随机插入 / 删除词、回译（EDA 数据增强）；
    - 特征层面：添加噪声、特征拼接、多模型特征融合（如 BERT+TF-IDF）。

# 不同场景特征处理选型建议

|任务场景|推荐特征处理方法|适用模型|
|---|---|---|
|短文本分类（如情感分析）|TF-IDF + N-gram / Sentence-BERT 嵌入|逻辑回归、SVM、轻量 NN|
|长文本分类（如文档分类）|LDA 主题特征 + BERT CLS 向量|Transformer、XGBoost|
|低资源场景（小数据集）|静态 Word2Vec + 统计特征|传统机器学习|
|高精度要求场景|预训练模型（BERT/RoBERTa）动态嵌入|深度学习模型|
