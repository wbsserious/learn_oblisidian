# 现代大模型的完整训练流程
```
原始数据
   ↓
第一步：预训练（Pre-training） 
   ↓
第二步：监督微调（SFT）
   ↓
第三步：偏好对齐（RLHF / DPO）
   ↓
第四步：（可选）后处理 / 安全过滤
   ↓
部署上线
```
# 核心标配技术点
- **架构**：Decoder-only（Llama/Qwen）、MoE（混合专家）
- **位置编码**：RoPE（旋转位置编码）
- **归一化**：RMSNorm
- **注意力**：GQA/FlashAttention 2
- **微调**：LoRA/QLoRA、SFT、DPO
- **推理**：vLLM、GPTQ/AWQ 量化、动态批处理
- **应用**：RAG、Agent、Function Calling
