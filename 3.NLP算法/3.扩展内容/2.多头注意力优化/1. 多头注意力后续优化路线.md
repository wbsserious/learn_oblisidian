多头注意力的优化沿着三个方向走：
1）结构变快（GQA/MQA）
2）位置变强（RoPE/ALiBi）
3）效率爆炸（FlashAttention）
# 一、结构优化（最核心）
## 1. **MQA 多查询注意力**
- 多组 Q，**共享 1 组 K、V**
- 速度极快，显存大幅降低
- 缺点：效果略降
- 代表：PaLM、早期 T5
## 2. **GQA 分组查询注意力（当前大模型标配）**
- 介于 MHA 和 MQA 之间
- Q 分成 G 组，**每组共享一组 KV**
- 速度 ≈ MQA，效果 ≈ MHA
- 代表：LLaMA 2/3、GPT-4、Qwen、Llama 3
## 3. **分组 / 滑动窗口注意力**
- 每个 token 只看局部窗口
- 长文本速度大幅提升
- 代表：Longformer、LLaMA 3、Mistral
# 二、位置信息优化（决定大模型上限）
## 1. **RoPE 旋转位置编码（绝对主流）**
- 对 Q、K 按位置旋转
- 注意力天然包含**相对位置**
- 无参数、外推强
- 代表：LLaMA、GPT-4、ChatGLM、Qwen
## 2. **ALiBi 偏置**
- 不用位置编码
- 直接加距离惩罚偏置
- 训练极稳、外推极强
- 代表：BLOOM、部分小模型
## 3. **T5 式相对位置偏置**
- 可学习的相对距离偏置
- 代表：T5、Flan-T5
# 三、效率优化（训练推理必备）
## 1. **FlashAttention v1/v2**
- 按块计算，降低显存
- 速度提升 2~5 倍
- **现在所有大模型必用**
## 2. **PagedAttention**
- 动态显存管理
- 超高并发推理
- 代表：vLLM
## 3. **线性注意力**
- 复杂度 O (n)
- 代表：Performer、Linformer
# 四、表达能力优化
## 1. **门控注意力（Gated Attention）**
- 加门控过滤信息
- 更稳、更深、表达更强
- 代表：多模态模型、混合架构
## 2. **解耦注意力（DeBERTa）**
- Q、K 位置编码解耦
- 理解类模型更强
## 3. **MoE 稀疏注意力**
- 多个专家头，只激活一部分
- 万亿模型必备
# 五、现代大模型**最终标配组合**
**GQA + FlashAttention 2 + RoPE + KV-Cache + Causal Mask**
