# 简介
**核心理念**:是**方向重于长度**。它用 `sign` 函数将复杂的梯度值压缩到 `{-1, 0, 1}`，只保留方向信息。
sign函数：$$sign(x)=
	\begin{cases}
	-1,&if\ x<0\\
	0,&if\ x=0\\
	1,&if\ x>0
	\end{cases}$$
# 算法更新规则
1. 动量法更新梯度：
	$m_t = β₁ * m_{t-1} + (1 - β₁) * g_t$
2. 更新参数：
	$θ_t = θ_{t-1} - η * (sign(m_t) + λ * θ_{t-1})$
# 核心优势
- **极高的内存效率**：Lion 只需维护一个动量，相比 Adam 显存占用**减少一半。对于训练百亿、千亿级别的大模型，这个优势是决定性的。
- **隐式的正则化效果**：`sign` 操作忽略了梯度的大小，可以看作一种**自适应梯度裁剪**，能防止梯度爆炸。同时，它引入的噪声有助于模型跨越尖锐的局部最小值，找到更平滑、泛化能力更好的解。
- **通信高效的分布式训练**：在分布式训练中，`sign` 可以将需要传输的梯度压缩成**1-bit 甚至更低精度的向量**，大幅降低通信瓶颈。
