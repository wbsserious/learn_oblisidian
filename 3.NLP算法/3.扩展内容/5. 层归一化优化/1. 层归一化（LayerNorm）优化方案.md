# 一、原始 LayerNorm
- 对**整个 hidden 维度**做归一化
- 带可学习缩放 γ、偏移 β
- 问题：
	- 计算略贵
    - 某些场景分布不稳定
    - 与残差配合深度模型仍会震荡
# 二、层归一化的**4 个核心优化**
## 1. **位置调整：Post-LN → Pre-LN（最重要）**
不是改 Norm 本身，而是**放的位置**：
- **Pre-LN**：Norm 放在子层**前面**
- 代表：BERT、LLaMA、GPT、几乎所有现代模型
- 作用：**深度网络训练极度稳定**
## 2. RMSNorm（现在大模型默认标配）
去掉均值，**只做方差归一化**，均方根归一化：
$$RMSNorm(x)=Var(x)+ϵ​x​⋅γ$$
- 去掉均值
- 去掉偏移 β
- 更快、更简单、效果更好
- **代表：LLaMA、GPT-3、GPT-4、Qwen、ChatGLM**
这是 LayerNorm 最成功的**简化优化**。
---
## 3. **分组归一化 GroupNorm / DeepNorm**
- 把特征分成几组，**每组单独 Norm**
- 稳定性更强，适合超深模型
- 代表：DeepNet、部分多模态大模型
---
## 4. **无偏移 / 无缩放变体**
- 去掉 β 或 γ
- 减少参数、加速计算
- 精度几乎不掉
- 代表：大量轻量化、高效大模型
---
# 三、现代大模型最终标配（背这个）
## **Pre-LN + RMSNorm**
LLaMA、GPT、Qwen、ChatGLM、Mistral 全用这套。
