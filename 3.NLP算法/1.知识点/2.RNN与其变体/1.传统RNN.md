## RNN的层次结构
![[Pasted image 20260225233510.png]]
1. 词嵌入层：将词转化为向量表示（词向量），参数为词嵌入矩阵或词向量矩阵。
2. RNN层：逐步循环处理词向量，生成每个时间步的隐藏状态。
3. 全连接层/输出层：将隐藏状态线性映射到输出，通常是词表vocab中的词的概率分布。
![[1.1RNN张量形状变化|100%]]
**循环连接 (Recurrent Connection)**：RNN的核心在于其循环连接，它允许**信息在网络内部循环流动**。这使得网络可以保留过去的信息，并将其用于处理当前的输入。
**隐藏状态 (Hidden State):** RNN在每个时间步都会维护一个隐藏状态，这个隐藏状态**包含了过去的信息**。它可以被认为是网络的”记忆”。
**时间步 (Time Step):** 指的是**序列数据中的每个元素**，例如文本中的一个单词、语音中的一个音频帧。

**模型优点**：
1. 天然适合序列数据建模。
2. 结构简单、参数少、计算轻量
3. 理论上具备记忆能力
	适合**短序列**任务
4. 奠定了序列模型的基础思想

**模型缺点**：
1. **严重的梯度消失/梯度爆炸**
	- 反向传播时梯度随时间步**指数级衰减或爆炸**。
	- 无法学习**长距离依赖**（句子长一点就记不住开头）
2. **长期依赖能力极弱** 
3. **训练极不稳定、难以收敛**
4. **无法进行并行计算
5. **容易过拟合且缺乏控制能力**
	- 没有门控，**无法自主选择记住 / 遗忘**。
	- 噪声、冗余信息都会累积在隐藏状态里。
## 模型分类
- 按照**输入和输出的结构**进行分类:
    - N vs N-RNN：输入序列和输出序列的长度相同，并且通常在时间步上是一一对应的。
    - N vs 1-RNN：输入一个长度为 N 的序列，输出一个单一的值或向量。
    - 1 vs N-RNN：输入一个单一的值或向量，输出一个长度为 N 的序列。
    - N vs M-RNN：输入一个长度为 N 的序列，输出一个长度为 M 的序列，其中 N 和 M 可以不相同。
