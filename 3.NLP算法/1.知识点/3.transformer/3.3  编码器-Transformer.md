**作用：**
把输入序列"理解"成一组高级语义特征的模块。
# 核心结构
编码器由**N**层完全相同的**编码器层**堆叠而成：
## **单层编码器结构（固定顺序）：**
1. **多头自注意力（Multi-Head Self-Attention）**
2. **残差连接 + LayerNorm**
3. **前馈网络（Feed Forward Network, FFN）**
4. **残差连接 + LayerNorm**
**公式简写**：
$$\begin{align}
x_1 &= \text{LayerNorm}(x + \text{MultiHeadAttention}(x)) \\ 
x_2 &= \text{LayerNorm}(x_1 + \text{FFN}(x_1))
\end{align}$$
## **多头自注意力（Multi-Head Self-Attention）**
**作用：** 让模型在多个不同的子空间里，同时、并行地捕捉输入序列中不同类型、不同距离的依赖关系，从而得到更丰富、更全面的上下文表示。
**Q, K, V**都是来来自于上一层的输出
核心作用：
1. 捕捉**全局依赖**
2. **多视角建模**（多头的意义）
3. **增强表达能力**
4.  **并行计算**
## 残差连接（Add）
**作用：** 解决深度神经网络的梯度消失问题，让模型可以堆叠得非常深，并且训练更稳定、更容易收敛。
	1. 防止梯度消失。
	2. 让深层网络可以训练。
	3. 加速收敛，稳定训练。
## 层归一化（LayerNorm）
[[10.正则化]]
作用：**层归一化是对单个样本的所有特征做归一化，把数据拉到均值 0、方差 1 的分布，让深度网络训练更稳定、更快收敛。**
	1. 稳定训练
	2. 加速收敛
	3. 降低对初始化敏感
#### 关键特点（和 Batch Norm 最本质区别）
- **Layer Norm：对 单个样本、所有维度 归一化**
    → 适合 NLP、Transformer、batch 大小变化的场景
- **Batch Norm：对 所有样本、单个维度 归一化**
    → 适合 CV，不适合变长序列
#### 公式:
$$\begin{align}
\mu=&\frac{1}d\Sigma^d_{i=1}x_i\\ \sigma=&\sqrt{\frac{1}d\Sigma^d_{i=1}(x_i-\mu)^2+\epsilon}\\
LayerNorm(x)=&\gamma·\frac{x-\mu}{\sigma}+\beta
\end{align}$$
- $γ$、$β$：可学习缩放偏移参数
- 均值、方差**只看当前一个样本**
#### 在 Transformer 里的位置
每一层都固定出现两次：
1. 多头自注意力后：
$$x=LayerNorm(x+Attention(x))$$
2. 前馈网络后：
$$x=LayerNorm(x+FFN(x))$$
## 前馈网络（Feed Forward）
**前馈神经网络（Feed-Forward Network, FFN）是对序列中每个位置**单独、独立**进行非线性变换的两层全连接网络，用来建模单个 token 内部的复杂语义特征。**
**作用**：
	1. **每个词自己的特征升级**
	2. 引入**非线性**，让模型能拟合复杂函数
	3. 对每个位置**独立处理**，不依赖其他位置
公式：
$$
FFN(x) = max(0,xW_1+b_1)W_2+b_2
$$
**FFN 在 Transformer 里的位置**
- 编码器：自注意力 → 残差 → LayerNorm → FFN
- 解码器：自注意力 → 残差 → LayerNorm → 交叉注意力 → 残差 → LayerNorm → FFN
