GPT（Generative Pre-trained Transformer，生成式预训练 Transformer）是 OpenAI 研发的**仅解码器架构（Decoder-only）** 大型语言模型（LLM）家族，核心是通过**自回归语言建模**的预训练范式，学习通用语言与世界知识，实现文本理解、生成、推理等能力，是当前自然语言处理领域的主流技术范式之一。
# 简介
GPT 是基于 Transformer 解码器的**生成式预训练语言模型**，核心目标是建模自然语言序列的概率分布，通过 “**预训练 + 微调 / 提示学习**” 的两阶段流程，从海量无标注文本中学习语言规律、事实知识与推理逻辑，最终具备通用的语言理解与生成能力。
**核心**
是通过**自回归语言建模**的预训练范式，学习通用语言与世界知识，实现文本理解、生成、推理等能力。
**技术本质**：**自回归概率模型**，给定前文序列，预测下一个 token 的概率分布，逐词生成文本。
**架构特性**
：**仅使用 Transformer 解码器**，通过**掩码自注意力**（Masked Self-Attention）保证生成时只能依赖前文，符合人类逐词表达的逻辑。
**核心范式**
**预训练（通用能力）+ 下游适配（任务能力）**，从 “专用模型” 走向 “通用模型”。
# 模型架构
GPT 的底层是堆叠的 Transformer 解码器层，标准结构如下：
## 输入层
- **词嵌入（Token Embedding）**：将离散 token 映射为高维向量（如 768/12288 维），捕捉词汇语义。
- **位置编码（Positional Encoding）**：为 token 添加序列位置信息（Transformer 无循环结构，无法天然感知顺序），常用正弦 / 余弦编码或可学习位置嵌入。
## 解码器层（核心计算单元，多层堆叠）
每层包含两个核心模块 + 残差连接与层归一化：
**掩码多头自注意力（Masked Multi-Head Self-Attention）**
- **自注意力**：计算每个 token 与序列中所有 token 的相关性权重，加权求和得到上下文表示，解决长距离依赖问题。
- **掩码机制**：强制模型在预测第i个 token 时，仅能访问第1到i−1个 token，保证自回归生成的合法性。
- **多头注意力**：将注意力拆分为多个并行子空间（如 12/96 头），分别捕捉不同类型的语义依赖（如语法、指代、逻辑），再拼接融合，提升表达能力。
**位置前馈网络（Position-wise Feed-Forward Network, FFN）**
- 两层全连接神经网络，对每个 token 的表示独立进行非线性变换（先升维后降维，常用 ReLU/GELU 激活），增强模型的非线性建模能力。
**残差连接与层归一化**
- **残差连接**：在注意力 / FFN 模块后加入输入直连，缓解深层网络的梯度消失 / 爆炸问题。
- **层归一化（Layer Norm）**：对每个样本的特征做归一化，稳定训练过程、加速收敛。
**输出层**
- 线性层 + Softmax：将解码器输出映射为词表维度的向量，经 Softmax 得到每个 token 的生成概率，选择概率最高的 token 作为输出（或采样）。\
# 预训练机制
- **目标函数**：**自回归语言建模（Autoregressive Language Modeling）**，最大化给定前文时下一个 token 的对数似然：
    $$
    \mathcal{L}=−∑_{i=1}^n​logP(xi​∣x_1​,x_2​,...,x_{i−1}​;θ)
    $$
    其中$θ$为模型参数，$x_i$​为第$i$个token。
- **数据与规模**：使用海量无标注文本（如书籍、网页、论文、代码），GPT-3 训练数据约 1.7 万亿 token，GPT-4 进一步扩展并加入多模态数据。
- **规模法则（Scaling Law）**：模型性能随**参数数量、训练数据量、计算量**的增加呈幂律提升，这是 GPT 系列迭代的核心理论基础。
# 微调范式
## GPT-1：监督微调（Supervised Fine-Tuning）
预训练后，在标注任务数据上微调模型参数，适配分类、问答、翻译等特定任务。
## GPT-2/GPT-3：零样本 / 少样本 / 上下文学习（In-Context Learning）
无需微调，仅通过**自然语言提示（Prompt）** 引导模型完成任务：
- 零样本：直接给出任务指令（如 “翻译：你好→Hello”）。
- 少样本：提供少量示例（如 “翻译：苹果→Apple；香蕉→Banana；橙子→？”）。
- 上下文学习：利用输入文本的上下文信息，动态适配任务，彻底颠覆 “预训练 + 微调” 范式。
## GPT-3.5/GPT-4：对齐训练（Alignment）
- **监督微调（SFT）**：用人类标注的高质量对话数据微调，让模型输出更符合人类意图。
- **基于人类反馈的强化学习（RLHF）**：训练奖励模型（RM）评估模型输出质量，再用 PPO 算法优化模型，提升有用性、安全性与一致性。
