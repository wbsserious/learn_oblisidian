# 基础介绍
在**已经预训练好的基座模型**基础上，使用**特定任务的标注数据**，以**较小的学习率**继续更新模型参数，使模型**适配并优化特定下游任务**的过程。
## 核心目标
将通用语言能力**迁移到具体任务**，例如：
- 文本分类
- 命名实体识别 NER
- 情感分析
- 机器翻译
- 问答系统
- 指令遵循、对话
## 训练方式
- 任务有明确标签（有监督学习）
- 通常只需要少量数据
- 学习率远小于预训练（避免破坏预训练知识）
- 可选择：全参数微调 / 部分参数微调
## 特点
- 数据量小、成本低
- 任务专一
- 模型在该任务上效果显著提升
- 通用性会下降（任务偏向性增强）
# 全参数微调 Full Fine-tuning
- 更新模型**全部参数**
- 效果最好
- 显存占用大
- 容易过拟合小数据集
# 高效参数微调 PEFT / Parameter-Efficient Fine-tuning
只微调**极少部分参数**，冻结基座：
- LoRA
- Adapter
- Prefix Tuning
- IA3
优点：
- 显存占用极低
- 训练快
- 不会灾难性遗忘
- 可插拔、多任务互不干扰
# 扩展
## 指令微调 Instruction Tuning
属于**微调的一种**，但任务更泛化：
- 用成千上万种任务的指令数据
- 目标不是单任务，而是**学会遵循人类指令**
## 人类对齐微调 RLHF / DPO
[[6. RLHF]]
也属于微调：
- 基于人类偏好
- 让模型更安全、有用、无害

大模型训练范式
基座预训练模型 → 微调 → 对齐模型
