# 预训练 Pre-training
**预训练**是指：
在**大规模通用无标注文本语料**上，使用**无监督 / 自监督学习目标**，训练一个**具备通用语言理解与生成能力的基础模型**的过程。
这个阶段得到的模型称为：
- **Pre-trained Model**
- **Foundation Model / Base Model**（基座模型）
# 核心目标
不是解决某一个具体任务，而是**学习语言本身的规律**：
- 语法、句法、语义
- 世界知识、常识
- 文本结构、逻辑关系
- 长距离依赖、上下文建模
# 训练方式（自监督任务）
模型利用文本自身构造监督信号，不需要人工标注：
- BERT：掩码语言模型 MLM
- GPT：自回归语言模型 LM
- T5：跨度破坏 Span Corruption
# 特点
- 数据量极大（万亿 tokens 级别）
- 计算成本极高
- 训练一次即可复用给无数下游任务
- 模型**通用但不针对具体任务**
