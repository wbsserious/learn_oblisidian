# 作用
1. 捕捉长距离依赖关系。
2. 并行性。
3. 灵活性和可扩展性强。
# 与RNN对比

| 维度        | 核心要点                              | 对比 RNN/LSTM        |
| :-------- | :-------------------------------- | :----------------- |
| **核心机制**  | 多头自注意力（Multi-Head Self-Attention） | 循环 / 门控机制，逐词串行     |
| **计算效率**  | 全序列并行处理，时间复杂度 O (n²・d)            | 串行处理，时间复杂度 O (n・d) |
| **长程依赖**  | 直接捕捉任意位置 token 关联                 | 随序列长度增加，捕捉能力衰减     |
| **标志性结构** | 编码器 - 解码器（Encoder-Decoder）        | 编码器 / 解码器多为循环结构    |

# Transformer 架构
## 1. 编码器（Encoder，6 层堆叠）
每层包含两个子层：
- **多头自注意力**：将输入映射为 Q、K、V，分 8 头并行计算注意力权重，再拼接输出，实现多视角语义建模。
- **前馈网络（FFN）**：两层全连接（中间 ReLU 激活），对每个位置独立变换。
- **残差连接 + 层归一化**：`LayerNorm(x + SubLayer(x))`，稳定训练。
## 2. 解码器（Decoder，6 层堆叠）
在编码器基础上新增**掩码多头自注意力**（Masked Multi-Head Self-Attention），防止解码时看到未来 token；并加入**编码器 - 解码器注意力层**，利用编码器输出的 K、V 与解码器的 Q 交互，实现源 - 目标对齐。
## 3. 输入 / 输出处理
- **词嵌入**：将 token 映射为 d_model=512 维向量。
- **位置编码**：通过正弦 / 余弦函数注入位置信息，解决并行处理的位置缺失问题。
- **输出层**：线性变换 + Softmax，生成下一个 token 的概率分布。
# 集中模型对比

| 项目         | **原始 Transformer**                   | **T5**                   | **BERT**               | **GPT**                       |
| ---------- | ------------------------------------ | ------------------------ | ---------------------- | ----------------------------- |
| **架构**     | Encoder + Decoder（完整结构）              | Encoder + Decoder        | **Encoder only**       | **Decoder only**              |
| **核心任务范式** | 机器翻译（Seq2Seq）                        | 统一 **文本→文本** 生成          | 掩码语言模型（MLM），**理解型**    | 自回归语言模型（AR），**生成型**           |
| **训练方式**   | 翻译监督训练                               | 降噪预训练 + 文本到文本微调          | 掩码预测 MLM + 句子关系预测 NSP  | 预测下一个词（Next-Token Prediction） |
| **注意力类型**  | Encoder: 全可见<br><br>Decoder: 掩码 + 交叉 | 同原版 Transformer          | 全可见自注意力（无掩码）           | 掩码自注意力（只能看左边）                 |
| **位置编码**   | 固定正余弦编码(绝对位置)                        | 相对位置编码（本质为**可学习相对位置偏置**） | 可学习位置编码(绝对位置)          | 可学习位置编码(绝对位置)                 |
| **擅长方向**   | 翻译、Seq2Seq 任务                        | 翻译、摘要、生成、多任务统一建模         | 分类、匹配、抽取、阅读理解等**理解任务** | 对话、续写、创作、摘要等**自由生成任务**        |
| **典型代表**   | 论文基准模型                               | T5-Large / T5-XXL        | BERT-base / BERT-large | GPT-2 / GPT-3 / GPT-4         |
| **一句话定位**  | 一切 Transformer 模型的**基础架构**           | 万能文本→文本生成模型              | 最强**语义理解**模型           | 最强**自由文本生成**模型                |
