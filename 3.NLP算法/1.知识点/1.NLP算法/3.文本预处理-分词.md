# 经典中文分词工具——jieba
**jieba（结巴）** 是基于 Python 开发的**开源中文分词工具**，是中文 NLP 领域最主流的分词库之一，核心定位为面向通用场景的高效中文词语切分工具。
## 分词原理
融合三种分词策略，兼顾精度和效率：
- 基于**前缀词典**的正向最大匹配（FMM）、反向最大匹配（RMM）
- 基于**隐马尔可夫模型（HMM）** 的统计分词（处理未登录词）
- 基于**Viterbi 算法**优化路径选择，提升分词准确性。
## 核心功能
- 基础分词：支持**精确模式、全模式、搜索引擎模式**三种切分模式；
- 词性标注：可输出分词结果对应的词性（如名词、动词、形容词）；
- 自定义词典：支持加载用户自定义词典，适配专业领域分词需求；
- 关键词提取：集成 TF-IDF、TextRank 算法实现关键词抽取；
- 并行分词：支持多进程加速，提升大规模文本处理效率。
## 三种分词模式对比

|模式类型|特点|适用场景|
|:--|:--|:--|
|**精确模式**|试图将句子最精确地切分，无冗余，适合文本分析|文本分类、情感分析、命名实体识别|
|**全模式**|把句子中所有可能的词都扫描出来，存在冗余|关键词提取、信息检索|
|**搜索引擎模式**|在精确模式基础上，对长词再次切分，提升召回率|搜索引擎分词、关键词匹配|

精确模式：试图将文本进行最精确的切割，适用于文本分析，情感分类，命名实体识别
全模式：试图将文本中的所有词都识别出来，会存在冗杂，适用于关键词提取，信息检索
搜索引擎模式：在精确模式切割的基础上，对长词再次切分，提高召回率，适用于搜索引擎分词，关键词匹配。

## **用户自定义词典**
jieba分词支持用户添加词典从而提高对专业领域文本的分词结果。
# 命名体识别（NER）
命名实体识别（NER）是自然语言处理中的一个任务，旨在从文本中识别出特定类别的实体（如人名、地名、机构名、日期、时间等）。NER是信息抽取的一部分，帮助计算机识别出与任务相关的实体信息。
- **同词汇一样, 命名实体也是人类理解文本的基础单元, 因此也是AI解决NLP领域高阶任务的重要基础环节.**
## 主要应用场景
NER 主要用于**从非结构化文本中自动识别关键实体**，广泛应用于**智能问答、信息检索、知识图谱、舆情分析、金融风控、医疗信息处理**等场景。
- **信息抽取**：NER帮助从海量的文本中自动抽取出结构化的实体信息，为数据分析、问答系统等提供有价值的内容。
- **问答系统**：在智能问答系统中，NER能够帮助系统准确理解用户的提问，并提取相关的实体信息以便生成更准确的回答。
- **文本理解**：NER对于文本理解至关重要，它帮助系统识别出文本中的关键信息，例如人物、地点、组织等，进而为语义分析和事件抽取提供支持。
## 常用方法
- **传统方法**：基于规则、词典、HMM、CRF
- **深度学习方法**：BiLSTM+CRF、BERT+CRF 等
# 在实际项目中，如果 Jieba 对专业领域词汇（如"黑马程序员"）分词效果不佳，你会如何解决？
1.添加自定义词典，调整专业领域词汇的词频。
2.若只有少部分专业领域词汇分词效果较差，可以手动对分词结果进行调整，或者在分词前对相应词语进行正则化处理。
3.结合相应专业领域预料对jieba分词模型进行分词处理。
4.结合其他分词工具共同对专业领域文本进行分词，比如引进HanLP（韩LP），结合NER优化分词结果。
5.如果为深度学习NLP算法，可以使用实体感知分词（Entity-aware segmentation）

# 词性标注（POS Tagging）
词性标注(Part-Of-Speech tagging, 简称POS)就是为文本中的每个词分配一个语法类别（即词性），例如名词、动词、形容词等。
## 常见方法：
- **传统方法**：基于规则、HMM、CRF、最大熵模型
- **深度学习方法**：BiLSTM、BERT 等预训练模型
## 作用：
- 为**句法分析、语义理解、信息抽取、NER**等上层任务提供基础特征。
- 帮助理解句子结构与词语间的语法关系。
- 提升文本分类、机器翻译、问答系统等任务效果。
