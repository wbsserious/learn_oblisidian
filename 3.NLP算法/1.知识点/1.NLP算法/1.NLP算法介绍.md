**核心定义**：**自然语言处理（Natural Language Processing, NLP）算法**，是**面向自然语言处理任务的一整套算法、模型与方法的统称**，
**目标**:是让**计算机**能够**理解、生成、翻译、处理人类语言（文本 / 语音转文本）**。
[[1.NLP算法]]
# NLP 任务标准处理流程
## 1. 数据准备
- 收集文本数据（爬取、标注、公开数据集）
- 划分：训练集 / 验证集 / 测试集
- 检查：是否有脏数据、标签错误、不平衡
## 2. 文本预处理（你刚才问的）
把 “人话” 变成 “模型能吃的数据”：
1. 清洗：去链接、特殊符号、噪声
2. 统一格式：大小写、繁简、编码
3. 分词（中文分词 / 英文分词）
4. 去停用词
5. 词形归一化（英文）
6. 构建词典、转成数字序列
## 3. 文本表示（把词变成向量）
让模型能计算：
- 传统：One-hot、TF-IDF、Word2Vec、GloVe
- 现代：BERT/RoBERTa 等预训练模型词嵌入
## 4. 模型搭建与训练
根据任务选模型：
- 分类：CNN、RNN、LSTM、Transformer、BERT
- 序列标注：BiLSTM+CRF、BERT
- 生成：GPT、T5、Bart
- 匹配：Siamese、BERT
流程：
- 定义模型结构
- 选择损失函数、优化器
- 训练、看指标（loss、acc、f1…）
- 调参、早停、防止过拟合
## 5. 评估 & 部署 & 应用
- 在测试集评估效果
- 错误分析 → 回头优化数据 / 模型
- 部署成 API / 服务
- 上线使用：搜索、推荐、对话、审核、摘要、翻译等
# 主要任务
- **语音识别**：将语音信号转化为文本
- **文本分析**：从文本中提取有意义的信息，包括情感分析、主题提取等
- **机器翻译**：自动将一种语言的文本翻译为另一种语言
- **语法分析**：分析句子的语法结构，识别句子中的各个成分（如主语、谓语、宾语等）
- **命名实体识别（NER）**：识别文本中的重要实体，如人名、地点名、组织名等
- **对话系统**：使计算机能够与人类进行自然流畅的对话，如智能助手、聊天机器人等
- **自动摘要**：从大量文本中提取出最关键信息并生成简洁的摘要
# 发展历史
## 1. 第一代：规则与符号主义 NLP 算法（非机器学习）
完全依靠**人工编写语言规则、词典、语法模板**，无自动学习能力，是 NLP 最早形态。
## 2. 第二代：传统机器学习 NLP 算法（浅层学习，非深度学习）
以**统计机器学习**为核心，依赖**人工特征工程**，模型本身是浅层结构，是深度学习普及前的工业主流。
## 3. 第三代：深度学习 NLP 算法（当前主流与 SOTA）
用深度神经网络自动学习语言的分布式表示，**端到端建模、弱化人工特征**，是现代 NLP 的核心。
按技术演进可分为：**词嵌入 → 序列网络 → Transformer 预训练模型 → 大模型时代进阶算法**。
## 4.当前与未来发展（2020s以后）    
 **大规模语言模型**：如GPT系列、BERT系列、T5、PaLM等，已经在文本生成、自动摘要、智能对话等任务上达到了接近人类的水平。
 **多模态学习**：越来越多的研究开始结合NLP和视觉、语音等其他模态，推动**跨模态理解**的研究。
 **低资源语言处理**：如何让机器理解低资源语言（例如少数民族语言、方言等）仍然是NLP的一个重要挑战。
**伦理和偏见问题**：随着模型越来越强大，NLP研究者也在关注模型的伦理性、透明性、公平性以及减少偏见的挑战。
